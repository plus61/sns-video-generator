## UI Color Scheme
- Background: Gray-100 (#F3F4F6)
- White containers with shadow
- Blue primary button (#2563EB hover: #1D4ED8)
- Green success elements (#10B981)
- Yellow warning box (#FEF3C7)
- Red error messages (#EF4444)
- Spinner: Blue-600 with transparent top

## Responsive Behavior
- Max width: 672px (max-w-2xl)
- Centered container with padding
- Mobile-friendly with full-width elements
- Video previews max height: 200px

## Accessibility Features
- Semantic HTML elements
- ARIA labels on interactive elements
- Clear focus states
- High contrast text
- Loading announcements for screen readers
